# super-rs8
(Super) Chip-8 emulator written in Rust
  
Usage:  
`super-rs8 <rom file>`  
  
Controls:  
```
123A 1234
456B QWER
789C ASDF
D0EF ZXCV

Escape: quit
Space: pause/resume
Enter: reset
Left/Right arrow: decrease/increase clock speed
Delete: register dump
```
